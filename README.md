# HTML Portfolio

A simple personal portfolio website built using HTML, CSS, and basic frontend concepts. This project is created to showcase personal details, education, skills, projects, and contact information in a clean web page format.

## Project Highlights

- Personal portfolio landing page
- Clean and simple layout
- Sections for skills, education, and projects
- Beginner-friendly HTML structure
- Easy to customize and improve

## Tech Stack

- HTML5
- CSS3
- Basic JavaScript

## Sections Included

- About Me
- Technical Skills
- Projects
- Education
- Contact Details

## What I Learned

- Creating structured web pages using HTML
- Styling layouts using CSS
- Building a personal portfolio for recruiters
- Organizing content for a professional profile
- Improving frontend basics through practice

## Future Improvements

- Add responsive design for mobile screens
- Add project screenshots
- Add animations and hover effects
- Connect contact form with backend
- Deploy using GitHub Pages
